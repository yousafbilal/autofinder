# 🚀 Quick APK Build Guide

## Problem Fixed! ✅

EAS CLI install ho gaya hai. Ab aap directly build kar sakte ho.

## ⚡ Sabse Tez Tarika (3 Steps):

### Step 1: Terminal mein ye command run karo:
```bash
cd AutoFinder-App
```

### Step 2: Build command run karo:
```bash
npm run build:android:apk
```

### Step 3: Jab prompt aaye, **"Y"** type karo aur Enter press karo

```
EAS project not configured.
? Would you like to automatically create an EAS project for @anasbakhan/autofinder? » (Y/n)
```

**Y** type karo aur Enter!

---

## 📱 Build Process:

1. **EAS project create hoga** (pehli baar)
2. **Build start hoga** (15-20 minutes)
3. **Email mein link aayega** ya terminal mein link dikhega
4. **APK download karo** aur install karo

---

## 🔧 Alternative: Manual Project Setup

Agar aap pehle project setup karna chahte ho:

```bash
eas build:configure
```

Jab prompt aaye, **Y** type karo.

---

## ✅ Build Complete Hone Ke Baad:

1. Terminal mein download link milega
2. Ya email check karo
3. Ya [expo.dev](https://expo.dev) pe login karke "Builds" section mein jao
4. APK download karo
5. Phone pe install karo

---

## 🆘 Agar Koi Problem Aaye:

1. **Permission error**: `eas login` karke correct account se login karo
2. **Build fail**: Error message check karo, usually dependency issue hota hai
3. **Timeout**: Internet connection check karo

---

**Note**: Pehli baar build 15-20 minutes lag sakta hai. Sabar rakho! 😊
